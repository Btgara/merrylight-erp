# 🆘 HELP & TROUBLESHOOTING

## ❓ COMMON QUESTIONS

### Q: Where do I download this?
A: All files are in the ZIP. Extract and use.

### Q: Do I need a database?
A: No! Everything works in your browser.

### Q: Will my data be saved?
A: Yes, in your browser. Don't clear cache or data loss happens.

### Q: Can others see my data?
A: No, data is only on your computer/device.

### Q: How do I back up my data?
A: Use Master Files export or screenshot important data.

## 🚨 PROBLEMS & SOLUTIONS

### Problem: "npm command not found"
**Solution:** Install Node.js from nodejs.org

### Problem: "git command not found"
**Solution:** Install Git from git-scm.com

### Problem: GitHub push fails
**Solution:** 
- Check your GitHub username is correct
- Check your email is correct
- Create personal access token if needed

### Problem: Vercel deploy fails
**Solution:**
- Check build logs in Vercel
- Look for error message
- Usually missing npm package
- Contact Vercel support

### Problem: Work Order dropdowns empty
**Solution:**
- Go to Master Files
- Add Customers, Varieties, Pack Types
- Click Save
- Refresh page

### Problem: Barcode not printing
**Solution:**
- Use Chrome or Firefox
- Check barcode format (CODE128)
- Test with online barcode reader

### Problem: Data disappeared
**Solution:**
- Don't clear browser cache!
- Use same browser/device
- If cleared, data is lost permanently

### Problem: Page not loading
**Solution:**
- Hard refresh: Ctrl+Shift+R
- Clear browser cache
- Try different browser
