# 🎉 MERRYLIGHT ERP - FEATURES

## 📊 DASHBOARD
Overview of all operations with real-time stats.

## 🍓 FRUIT RECEIVING
Manage incoming fruit batches:
- Upload batches with harvest details
- Print barcodes (CODE128 format)
- Track net weight (automatic calculation)
- Filter by variety, supplier, status

**Print Label Includes:**
- Batch number (large, bold)
- Scannable barcode
- Variety, block, weight, harvest date

## 📦 PACKING OPERATIONS (5 TABS)

### Tab 1: Work Order Creation
Create customer orders:
- Auto-generated order numbers
- Select Customer (from Master Files)
- Select Variety (from Master Files)
- Select Pack Type (from Master Files)
- Set target quantity & weight
- Track status

### Tab 2: Tip to Work Order
Link fruit batches to orders:
- Select work order
- Select fruit batches (multi-select)
- Bulk assign to order
- Track which fruit goes where

### Tab 3: Pallet Registration
Register pallets:
- Auto-generated pallet numbers
- Link to work order
- Record quantity & weight
- Track pallet type
- Set status (Building → Full → Ready)

### Tab 4: Pallets Report
View all pallets:
- Complete pallet listing
- Filter by status
- See all details
- Track dates

### Tab 5: Packout Summary
Operations overview:
- Total work orders
- Active orders
- Total pallets
- Ready pallets
- Total quantity packed (units)
- Total weight packed (kg)

## ⚙️ MASTER FILES (15 TYPES)

**Important for Work Orders:**
- **Customers** - Your customers (used in dropdown)
- **Varieties** - Your fruit types (used in dropdown)
- **Pack Types** - Your packing formats (used in dropdown)

**Other Master Data:**
- Sections - Farm sections
- Blocks - Farm blocks
- Commodities - Commodity types
- Pallet Types - Pallet formats
- Crate Types - Crate formats
- Grades - Fruit grades
- Seasons - Season years
- Channels - Sales channels
- Transport - Transport providers
- Sizes - Package sizes
- Brands - Product brands
- Invoice Codes - Invoice codes
