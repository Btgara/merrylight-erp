# 🚀 QUICK START - DEPLOY IN 5 MINUTES

## STEP 1: Extract ZIP
- Right-click ZIP file
- Click "Extract All"
- Done! ✅

## STEP 2: Open PowerShell
- Press Windows key
- Type "PowerShell"
- Click "Windows PowerShell"
- Copy-paste this:

```bash
cd C:\Users\User\Downloads\merrylight-erp
```

Press Enter.

## STEP 3: Create GitHub Account
- Go to https://github.com/signup
- Create account (free)
- Remember your username!

## STEP 4: Push to GitHub
Copy-paste each command ONE BY ONE:

```bash
git init
```
Press Enter.

```bash
git config user.name "Your Name"
```
Press Enter.

```bash
git config user.email "your@email.com"
```
Press Enter.

```bash
git add .
```
Press Enter.

```bash
git commit -m "Initial commit"
```
Press Enter.

```bash
git remote add origin https://github.com/YOUR_USERNAME/merrylight-erp.git
```
Replace YOUR_USERNAME with your GitHub username. Press Enter.

```bash
git branch -M main
```
Press Enter.

```bash
git push -u origin main
```
Press Enter and wait.

## STEP 5: Deploy on Vercel
1. Go to https://vercel.com/signup
2. Click "Continue with GitHub"
3. Authorize it
4. Click "Add New" → "Project"
5. Find "merrylight-erp" repository
6. Click "Import"
7. Click "Deploy"
8. Wait 2-3 minutes

## STEP 6: Your Live App!

After deployment, you'll see:
```
https://merrylight-erp.vercel.app/
```

**DONE!** 🎉

## STEP 7: Set Up Master Files
1. Go to your live app URL
2. Click "Master Files"
3. Add Customers
4. Add Varieties
5. Add Pack Types
6. Save all

## STEP 8: Start Using!
- Create fruit batches
- Create work orders
- Link fruit to orders
- Build pallets
- Track everything!

---

## ❓ NEED HELP?

### If GitHub push fails
- Check you replaced YOUR_USERNAME in step 4
- Check your GitHub username is correct

### If Vercel deploy fails
- Check build logs
- Usually missing dependencies
- Contact support if stuck

### If data not showing
- Go to Master Files
- Add Customers, Varieties, Pack Types
- Refresh page
