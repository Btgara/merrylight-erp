# 🍓 Merrylight Packhouse ERP

Complete Enterprise Resource Planning system for Merrylight Enterprises packhouse operations.

## ✨ FEATURES

### 📊 Dashboard - Real-time overview
### 🍓 Fruit Receiving - Batch management with barcode printing
### 📦 Packing Operations - 5 integrated tabs:
  - Work Order Creation (Customer, Variety, Pack Type from Master Files)
  - Tip to Work Order (Link fruit batches)
  - Pallet Registration (Register pallets)
  - Pallets Report (View all pallets)
  - Packout Summary (Operations metrics)
### ✅ Quality Control - QC checks
### 👥 HR Management - Employee records
### ⚙️ Master Files - 15 types of system data
